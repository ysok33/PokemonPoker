HUD PROFILE GUIDE

1. スタット定義

スタット 意味
VPIP 自分が任意でポットに参加した割合（参加ハンド数 ÷ 総ハンド数）。どれだけ広く / 狭くハンドをプレイしているかを示す。
PFR 自分からレイズでポットに入った割合。どれだけレイズで参加して主導権を取れているかを示す。
3bet％ 相手のオープンレイズに対して3betした割合。プリフロップでどれだけ積極的にプレッシャーをかけているかを示す。
Flop C-bet％ プリフロップレイザーとしてフロップを迎えた場面でCベットを打った割合。フロップでどれくらい攻めるかの指標。
Turn C-bet％ フロップ後もアグレッサーとしてターンを迎えた場面で2ndバレルを打った割合。ターン以降もバレルできているかの指標。
River C-bet％ ターンまでアグレッサーとして残った場面でリバーに3rdバレルを打った割合。バリューbetやブラフを最後まで打ち切れているかの指標。
Fold to Flop C-bet％ 相手からフロップでCベットを打たれたときフォールドした割合。フロップ以降どれくらい簡単にポットを諦めるかの指標。
AF （ベット＋レイズ）÷ コールで計算される攻撃性指標。全ストリートを通じてどれくらい自分から仕掛けるプレイヤーかを示す。
WTSD％ フロップ到達ハンドのうちショーダウンまで行った割合。どれくらいショーダウンまで粘るタイプかを示す。
W$SD％ ショーダウンに行ったハンドのうち勝った割合。ショーダウンまで行ったときにどれくらい勝てているかを示す。
WWSF％ フロップを見たハンドで最終的にポットを取った割合（ショーダウンの有無は問わない）。フロップ以降の取り合いの強さを示す。

2. プリフロップ分類（VPIP・PFR・3bet％）

使用スタット：VPIP / PFR / 3bet％ / 補助指標 PFR ÷ VPIP

Preflop データ不足 / ノーマル

RANGE OVERVIEW
データ不足
  • VPIP が「-」など未入力の場合

→ サンプル数が少ない / データ欠損のため、プリフロップのタイプ判定を行わない。

ノーマル
  • 下記 4タイプ（TAG / TPS / LAG / LPS）のいずれにも当てはまらない場合

→ VPIP / PFR / 3bet％ が極端すぎず、バランスが取れた平均的なプリフロップレンジとアグレッション。

目安：Zoom 6max なら VPIP 19〜24 / PFR 16〜21 付近など

Preflop TAG（タイトアグレ）

VALUE ORIENTED
条件
  • VPIP < 22％
  • VPIP ≠ 0 のとき PFR ÷ VPIP ≥ 0.70
  • 3bet％ ≥ 5％

→ 参加レンジはタイトだが、参加する時はほとんどレイズ・3betで入り主導権を取りに行く、典型的なタイトアグレタイプ。

→ 基本的に 強いレンジでポットを大きくしにいくため、ブラフよりもバリュー重視の傾向が強い。

VPIP 15〜21％ が出やすい PFR 12〜18％ が出やすい

Preflop TPS（タイトパッシブ）

CALL HEAVY
条件
  • VPIP < 22％
  • かつ以下のいずれか
    • PFR ÷ VPIP < 0.60
    • 3bet％ < 3％

→ 参加レンジは狭いものの、レイズよりコールが多く、3betも少ないため、プリフロップで受け身になりやすいタイプ。

→ オープンの頻度が低く、リンプ・コールでポットに参加しやすいプレイヤー像に近い。

Preflop LAG / LPS（ルース系）

LOOSE PROFILE
LAG（ルースアグレ）
  • VPIP ≥ 22％
  • VPIP ≠ 0 のとき PFR ÷ VPIP ≥ 0.65
  • 3bet％ ≥ 6％

→ 広いレンジでポットに参加し、その多くをレイズ・3betで入っていく攻撃的なルースアグレタイプ。

LPS（ルースパッシブ）
  • VPIP ≥ 22％
  • かつ以下のいずれか
    • PFR ÷ VPIP < 0.55
    • 3bet％ < 3％

→ 参加レンジは広いが、レイズよりもコールでついていくことが多く、3betもほとんどしないルースパッシブ（コーリングステーション寄り）。

LAG：VPIP 25％＋ / PFR 20％＋ が出やすい LPS：VPIP 25％＋ だが PFR がかなり低め

3. フロップ以降のC-bet・コール傾向

使用スタット：Flop C-bet％ / Turn C-bet％ / River C-bet％（ロジックでは未使用） / Fold to Flop C-bet％

Postflop データ不足 / 標準

FLOP PROFILE
データ不足
  • Flop C-bet％ / Turn C-bet％ / River C-bet％ / Fold to Flop C-bet％ のいずれかが未入力

→ C-bet や対C-betの頻度に関するデータが足りないため、フロップ以降のタイプ分類は行わない。

標準
  • 下記 4タイプ（打ち逃げ / バレルマシン / 受けすぎ / フォールドしすぎ）のいずれにも当てはまらない場合

→ Flop/Turn C-bet％ と Fold to Flop C-bet％ が極端でなく、平均的なC-bet頻度と守備傾向。

Postflop フロップC-bet打ち逃げ

ONE-AND-DONE
条件
  • Flop C-bet％ ≥ 55％
  • Turn C-bet％ ≤ 50％
  • Turn C-bet％ ≤ Flop C-bet％ − 10％

→ フロップではよくC-betを打つが、ターン以降のバレル頻度が明確に落ちる「1発打って諦めやすい」タイプ。

→ フロップでのコール＋ターンでのプレッシャーが有効になりやすい相手像。

Postflop バレルマシン

HIGH PRESSURE
条件
  • Flop C-bet％ ≥ 60％
  • Turn C-bet％ ≥ 50％

→ フロップ・ターンと高頻度で連続バレルするタイプ。ブラフ・バリューともに多く、常にプレッシャーをかけてくる。

→ 強いレンジでコールダウンし、リバーブラフキャッチを狙いやすい相手。

Postflop 受けすぎ / フォールドしすぎ

VS C-BET
受けすぎコーラー（コールダウン多）
  • Fold to Flop C-bet％ < 40％

→ 相手のフロップC-betに対してなかなかフォールドせず、コールに寄りがちなプレイヤー。ブラフに対しても簡単には降りない。

フォールドしすぎ
  • Fold to Flop C-bet％ > 70％

→ 相手のフロップC-betに対してかなりの頻度でフォールドしてしまうタイプ。こちらのフロップCBブラフが通りやすい相手。

受けすぎ：バリュー多めに打つ フォールドしすぎ：ブラフCB増やす

4. ショーダウン傾向（AF・WTSD％・WWSF％・W$SD％）

使用スタット：AF / WTSD％ / WWSF％ / W$SD％

Showdown データ不足 / バランス型

SD PROFILE
データ不足
  • AF / WTSD％ / WWSF％ / W$SD％ のいずれかが未入力

→ ショーダウン周りの傾向を判断するには、ハンド数やデータが不足している状態。

バランス型
  • 下記 2タイプ（ショーダウン頼み / ブラフ多め・スティール多め）の条件どちらにも当てはまらない場合

→ ショーダウン頻度・勝率・AF が極端に偏っておらず、ブラフとバリューのバランスが取れている平均的なショーダウン傾向。

Showdown ショーダウン頼み

SD HEAVY
条件
  • WTSD％ ≥ 33％
  • WWSF％ ≤ 53％
  • W$SD％ ≤ 46％
  • AF < 2

→ ブラフなどで途中のポットをあまり取らず、ショーダウンまで行きがちだが勝率も高くない「ショーダウンまで粘りすぎる」受け身タイプ。

→ バリュー薄めでもしっかり打つことで、オーバーコールを誘いやすい相手像。

Showdown ブラフ多め・スティール多め

AGGRO THREAT
条件
  • WTSD％ ≤ 31％
  • W$SD％ ≥ 49％
  • AF > 2.3

→ ブラフやスティールでショーダウン前に多くのポットを奪うため、WTSD％は低いが、ショーダウンまで行くときは強いハンドが多く勝率が高いアグレッシブタイプ。

→ ブラフキャッチの頻度を慎重にコントロールし、リバーでの大きなアクションには強いハンドで対抗したい相手。
